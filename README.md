# Local LLaMA Chatbot

A simple offline chatbot using **LLaMA 3B** and **Ollama**, with **Streamlit UI** for interaction.

## Features
- Text-based conversation
- Multi-turn chat using Streamlit `session_state`
- Runs locally without external servers
